What-O-Saur is a desktop-website that teaches kids about the dinosaur orders through animation.

View Website

Objective

Create a desktop website about the different dinosaur orders for children ages 5-10 with imagery and content is engaging and informative.

Results

What-O-Saur is a multi-page desktop website that is visually interactive through animations, buttons, and sounds. The written content showcases facts about the featured dinosaurs in a way that speaks directly to the viewer.

Wireframe

I originally intended for the site to consist of two full-screen pages that lead to a long, scrolling page dedicated to each respective suborder of dinosaurs and their facts.

I then realized that a more effective design to keep kids engaged, is to create multi-full-screen pages with buttons. I therefore altered my design to the final UI, which mostly eliminates the scroll-feature altogether.

Artwork

Sketches

Illustrator

I illustrated the monochromatic, cartoon dinosaurs to create welcoming, kid-friendly characters, which set the art-style for this site. I proceeded to create the rest of the content for the website based off of my research to create recognizable figures and icons.

After Effects

I animated each dinosaur to react to their surroundings or with the viewer. I created a slightly different animation for each character to reflect thier personality.
